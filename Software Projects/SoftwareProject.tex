\section*{Introduction}
\raggedright
The goal of this project is to create a program that plays a random selection of songs from a folder using the Pygame library. The program allows the user to skip to the next song by pressing a key.
\section*{Libraries}
The code utilizes the following libraries:
\begin{itemize}
    \item \texttt{pygame}: Provides functionality for multimedia applications such as playing audio.
    \item \texttt{numpy}: Used for generating random numbers.
\end{itemize}

\section*{Function: \texttt{song\_list()}}
The \texttt{song\_list()} function performs the following steps:
\begin{enumerate}
    \item Initializes an empty list to store the played songs.
    \item Initializes the Pygame mixer module for audio playback.
    \item Enters a loop to select and play random songs.
    \item Generates a random number to represent the index of an MP3 file.
    \item Checks if the generated number has already been played.
    \item Loads and plays the selected MP3 file.
    \item Enters a nested loop to wait for the song to finish playing.
    \item Prompts the user to press 's' to skip to the next song.
    \item If the user presses 's', stops the current song and breaks out of the loop.
    \item Prompts the user to press 'q' to quit the current playlist and start over again.
    \item If the user presses 'q', stops the current song and the main function calls the song\_list() function again.
    
\end{enumerate}

\section*{Main Loop}
The \texttt{song\_list()} function is called in a loop infinetely many times, creating a playlist of random songs.

\section*{Project Evaluation}
The code successfully creates a random song playlist by selecting and playing MP3 files from a specified folder. The user has the option to skip to the next song by pressing 's'. The program utilizes the Pygame library for audio playback and the NumPy library for generating random numbers.

\section*{Future Improvements}
There are several potential improvements for this project:
\begin{itemize}
    \item Add functionality to pause and resume the currently playing song.
    \item Implement error handling for incorrect user input.
    \item Allow the user to specify the number of songs in the playlist.
    \item Enhance the user interface with a graphical interface or command-line options.
\end{itemize}
